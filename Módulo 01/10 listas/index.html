Trabalhando com Listas

Listas Ordenadas

Para utilizar lista ordenada utilizaremos a tag (ol) ela irá organizar de forma automatica caso você preciso mudar a ordem

Para organizar a lista com numeros, ponto, numero romano utilizamos a tag (type dentro da tag ol)

Ex: ( ol type="1" )

  1. Acordar
  2. tomar caé
  3. escovar os dentes
  4. lavar Roupa

Listas Não Ordenadas

para as listas não ordenadas utilazamos a tag: (ul), pode ser usado dentro da ul o parametro type para definir o tipo de inicio da lista, pode ser marcado como: ( disc, circle square) ex: (type="square")

    Formato disc

  • pão
  • arroz

    Formato square

  • pão
  • arroz

    Formato circle

  • pão
  • arroz

Para juntar e organizar as listas antigas e novas citado no exemplo acima de forma crescente por numero ou letra utilizaremos a tag start dento da tag ol ex: (ol type="a" start="4") Note que é preciso contar onde para o numero da primeira lista e adicionar o proximo numero na tag (START) conforme no exemplo a 1º lista vai até ao 3º item e adicionamos o numero 4 para a (START)
